## About

This is my personal project for a great knowledge base with everything I know about Bioenergetics and it's relations.

It will have a PostgreSQL database with very detailed information about Vitamins, Aminoacids and other things related.

The idea is to have everything catalogued for easy searches, like a really big Graph, a Great Palace of the Mind.
